Licence Renewal Type
Overview

In the product details screen, it is possible to specify licence details (where applicable). For instance, if the product is software related, a licence code may be required periodically in order for it to remain active.

This field allow the specification of the type of renewal (e.g. Annual, Machine Specific).

Required Information

This is a single textual definition. The definition must be unique, hence the system will reject any duplicate definition supplied.

Once a definition has been provided, click the update button to commit it to the database.
