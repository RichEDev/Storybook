User Field Groupings
Overview

User Field Groupings are the headings that are required in order to create "collections" of user defined fields pertaining to a common topic or theme.

In order for these groupings to be displayed selectively, contract categories should be defined in advance if possible. If they are not, these grouping definitions will have to be revisited as the default is that the groupings are deselected for all categories.

Required Information

The following elements of information are required as part of the definition detail: -

Field Required Description
User Field Grouping This field is the title or name of the grouping that you wish to collate the user defined fields by.
Sequence The sequence field allows the explicit order of the groupings to be defined. All groupings will be sorted by sequence prior to being output.
Grouping Area Grouped user defined fields can be created in three main areas.
  • Additional Contract Details
  • Contract Products
  • Recharge Facility (assuming this is activated)
    		•
    Category to Display for? A list of contract categories are displayed in a table.

    Ticking the box at the side of each category will mean that, for every contract tagged as belonging to that category, will have this grouped collection of fields displayed.

    This allows only those fields relevant to particular categories of contracts being display, thus hiding those fields that may cause confusion.

    If many fields are defined, a barage of fields being displayed to the user can be therefore be avoided.

    Note: 'Select All' and 'Deselect All' buttons are provided to speed the process where many categories are defined.
