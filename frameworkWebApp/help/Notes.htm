Free Text Notes
Overview

Free text notes are stored relative to their context. Depending from where the Notes are accessed, they will be stored as one of the following types of note: -

  • Contract Note
  • Product Note
  • Vendor Note
  • Invoice Note
  • Vendor Contact Note

    • To improve the meaningfulness of the notes, each individual notes can be tagged with a primary and secondary definition. These are labelled as Note Category and Note Type.

    When a Note Category is selected, the Note Types will change to those associated with the selected type. For example, a Note Type of 'Recent Communication' may have Note Categories of 'Email','Letter' or 'Telephone'

    The Note Category and Types are definable by the user. Note Categories are available to user with access to the Table Maintenance.

     Adding a Note

  • Click the 'New' button

  • Select the Note Category and Note Type definitions as appropriate

  • Enter the date for the Note. This will default to the current date.

  • Type the note (or paste the required text from your clipboard) in to the note area.

  • Click the 'Update' button once the Note is completed.

    •  View / Edit an Existing Note

  • Existing Notes are displayed in the left hand panel ordered by Date and Note Category

  • Simply click the Note item you wish to view in the Note List panel

  • The note will then be loaded in full where it can be read or modified.

  • To save any modifications, click the 'Update' button.

  • To exit without making any modifications, click the 'Close' button.

    •  Delete a Note

  • Select the Note you wish to delete from the Note List pane on the left side of the Note screen.

  • A Note is selected by clicking in the list. This will then be loaded into the edit panel in full.

  • Click the 'Delete' button.

  • You will be prompted to confirm that you wish to proceed with the Note deletion.

  • Click 'OK' to confirm deletion.

  • The Note will then be deleted permanently from the database.
