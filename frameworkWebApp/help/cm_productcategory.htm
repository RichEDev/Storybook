Product Category
Overview

Product Category definitions are utilised within the product definition screen. Any product defined within the system can be assigned to a particular category.

By allocating products to a category, reports can be generated using this additional level of granularity.

Required Information

Product Category is a single text definition field. Each definition must be unique and any duplicate definition supplied will be rejected.

Once the definition has been specified, click the 'Update' button to commit the definition to the database.
