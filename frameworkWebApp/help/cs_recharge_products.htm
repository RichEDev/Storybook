Recharge Products
Overview

The recharge products tab displays all contract products available on the active contract that can be specified for recharging to another party.

Additional, recharge specific, information can also be specified for each of the recharge products in the form of user defined fields. These user defined fields can be individually defined or collated under a common group heading.

Grouped User Defined Fields

If specific information is created against the recharge products with user defined fields, these will be indicated on the screen by a button.

Clicking this button will expand the additional information into view. By collapsing this information in the first instance, the screen can be kept tidy and the potential for flooding the screen with information is reduced. The user can decide if they wish to see the fields or not for a given recharge product.
