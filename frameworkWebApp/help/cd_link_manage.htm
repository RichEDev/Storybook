Manage Contract Links
Overview

The contract link management screen allows the user to display which contracts are linked to which link definition.

Not only is it possible to review the contracts linked to each defintion, individual contract can be unlinked or the definition as a whole can be deleted, thus removing all it's links at the same time.

To Add a New LinkDefintion To simply create a new definition, without assigning any contracts to it, click the Add New Link Definition under the Contract menu option.
To Add a New Definition & Contract Link A contract link definition can be created at the same time as the adding of link. This action is triggered from the core contract detail screen.

Functions
Function Description
Link Definition

The drop down list contains all defined link definitions. By default, [None] is displayed, and as a result the will not be any contract information displayed.

Selecting a link definition from the list will display all contracts currently associated with that definition.

Selecting the hyperlink definitions of those contracts returned, will transfer you to either the contract or vendor (depending which definition was selected.
Delete Definition

If you elect to delete the actual link definition, you will be prompted to confirm this course of action. Clicking OK acknowledges your wish to proceed.

WARNING: Electing to delete a definition will result in the loss of all contract links associated with the definition.
Delete

For each contract displayed, the delete hyperlink allows it's deletion.

This provides the ability to remove a single contract link from a collection.
New Definition

If a new link definition is required, a text box appears at the top of the screen where the name can be specified.

On entering the definition, click the OK button and the new definition will be created.

If a contract link is specified, then the new link will be associated and displayed in the Link Management screen with any other linked contracts that have been previously linked.

If only a definition is being created, the user is returned to the Summary Portal after the definition has been created.

NOTE: Duplicate Link Definitions are not permitted. An error status message will inform of any encountered duplication. If this should occur, the definition will have to be edited before it can be used.
