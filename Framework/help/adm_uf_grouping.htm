User Field Groupings
Overview

Framework allows an unlimited number of User Defined Fields to be created across a number of screens within the application.

For increased clarity, especially within the Additional Contract Detail tab, these fields can be grouped together under common headings.

This area of the administration menu, is where these Groupings are defined.

On entering the screen, a list of existing definitions will be displayed.

  • To Add: Click the 'New' button at above the definition list. You will be transferred to the definition screen where the details of the element can be entered.

  • To Edit: Click the 'Edit' button to the left of the definition. You will be transferred to the definition screen where the detail of the element can be modified.

  • To Delete: Click the 'Delete' button to the left of the definition. You will be prompted to confirm deletion. Clicking 'OK' will remove the item from the list.

    • Note: User Field Groupings must be defined before the User Defined Fields are created.
