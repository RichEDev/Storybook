Framework Reports
Overview

The Framework report screen is where the executed report actually appears.

From this main report screen, the report can be viewed, exported, printed or modified (assuming the current user is the contract owner).

Note: A Global report can only be modified by the user who created it. This prevents modification of what might be considered 'standard' reports available for general use.
 Publish a Report as Global

Why Publish a Report as Global?

By default, all reports created are Personal and so can only be viewed or used by the user that created it. If a report you have compiled is thought to be of use to others, it can be published as Global enabling all other users to run it.

How do I publish my report?

To Publish the report, simply click the radio button at the top of the screen from Personal to Global.

This will change the status of the report.

Note: As 'creator' of the report, only they will be able to change the criteria, column selections etc.

Important: If the Global radio button is greyed out, the user does not have sufficient access rights to publish Global reports.
 Grouping By Column

What is Grouping?

Framework's reporting displays in the same way as Microsoft Outlook email application.

For added clarity, it may be desirable to view the returned report data by a particular column (e.g. Contract Vendor).

How do I group by a column?

Simply drag and drop the desire column header into the area above it marked 'Outlook Group by area'.

If you don't know how to drag and drop, follow the steps below: -

  • Using the mouse, left-click (and hold down) over the column header that you wish to group by.

  • While still holding the mouse button down, move the mouse into the area above the column headers. You should see the column header being dragged

  • Release the mouse button

    Tip: Select a column that has been sorted as your Grouping header for more meaningful results.

    Any column groupings you make will be stored with the report and retained for the next time the report is run.

    •  Export Report

    At the top right corner of the reporting screen, a drop-down list provides the ability to export the report in three ways. Select desired export and click the 'Export' button adjacent to it and a new window (printer friendly without menus) will re-run the report

    The three export types are: -

    Printer Displays the report in a format ready for printing. Type CTRL+P or choose Print from the Internet Explorer File menu. Remember to print using Landscape orientation as this is most likely to fit your columns to the page.
    Excel Microsoft Excel is a commonly used spreadsheet application and by providing an export, in this format, allows the flexibility to transform the report for other business documents. This will especially be the case for the financial reports it can create.

    Once opened in the export window, click the 'Export' button at the top of the screen. This will then export the Excel file.
    Comma Separate File (CSV) On exporting, a text file is generated and opened in a browser window. A CSV file is a common format for importing the data into other applications (e.g. Excel, MS Access etc.).
