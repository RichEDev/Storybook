Invoice Frequency Type
Overview

Invoice Frequency Type is assigned in the core contract detail screen. It allows specification of how often maintenance is invoiced to against a contract.

In the Invoice Forecasts screen, future payments can be automatically generated. The number of payments generated will be dictated by the Invoice Frequency selected on the core contract detail screen.

Required Information

The primary field is the definition by which the Invoice Frequency Type is to be known.

Against the definition a number of months is specified (i.e. 1 for Annual or 3 for Quarterly). This field must be numeric as it is used in Invoice Forecast generations.

Once a definition has been provided, click the update button to commit it to the database.
