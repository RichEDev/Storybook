Supplier Details
Overview

The term Supplier refers to the provider of the goods or services that have been obtained and the party with whom you are holding a contract with. It is used as a generic reference throughout this application.

Suppliers must be defined prior to being able to add a contract to the system.

This screen allows all the relevant information about the supplier to be entered.

The Address information on this screen refers to 'Primary' address. Different addresses can be specified against supplier contacts as and when they are added.

Finding a supplier
Tip: By far the quickest way to locate a supplier is via the Home page of Framework.

On selecting one of the returned suppliers from the search, the supplier and it's details will be automatically selected on entry to this screen.

Due to the typically large number of suppliers held within the system, Framework groups them alphabetically, displaying three letters at a time in the tabbed list to the left of the screen.

To change the letters displayed in the tabs, click the letter links that run horizontally above the supplier fields.

Clicking between the tabs, displays all suppliers whose names begin with the tab letter are listed.

Once the required supplier has been found, it can be selected simply by clicking on the name. The associated supplier information will then be loaded.

Note: Double clicking on a Supplier Details based report will also navigate to the Supplier Details screen and load the relevant Supplier information.
 Mandatory Fields

Most fields within Framework are optional. The more detail entered against a supplier, the more meaningful the data is. The following exceptions to the rule are mandatory fields. These are detailed below.

Field Name Definition
Supplier Name

The supplier name should be both unique and can be the formal name of the supplier company or, for user-friendliness, be the name by which they are better known to your organisation (assuming it to be different).

The field is text and therefore alphanumeric.

Modifying Supplier Information

The Supplier Details screen is where information about the supplier can be added, modified or deleted.

Action Method
Adding a Supplier

At the top left corner of the screen, a 'New' link or button is available to those users with adequate permissions.

On clicking this link, a blank screen is displayed and the fields can be filled in accordingly. Use the 'Tab' key to advance to the next field in the sequence.

Once all information has been entered satisfactorily, click the 'Update' link or button and the supplier will be added to the system.

NOTE: The supplier name is the primary key field and therefore must be unique. If the name is defined elsewhere, the addition will be rejected.
Delete a Supplier

At the top left corner of the screen, a 'Delete' link or button is available to those users with adequate permissions.

On clicking the link or button for deletion, the user is always prompted to confirm that this is the action they wish to take. Click OK to continue or Cancel to abort your action.

Before electing to delete a supplier, it must be the currently selected supplier with all it's details displaying on the screen.

Framework enforces referential integrity of the database at all times. This means that if the supplier is assigned to a contract within the database, the deletion will be aborted.

If the supplier is not assigned to a contract, the deletion will take place and all accompanying Contacts, Notes and Attachments will also be removed from the system.
Update a Supplier

Once a supplier has been selected for editing, the user is free to update the required information on the screen.

Once all modifications have taken place, click 'Update' to commit these changes to the database.

Note: Failing to click the update button prior to leaving the screen or selecting another supplier will result in loss of the modifications.

 Related Information

The following subset of information is associated with the Supplier Details.

These can only be added once a supplier exists on the database and when it is selected. Without a currently selected supplier, this information is not available.

Link or Button Definition
Supplier Contacts

When dealing with a supplier, many individuals from that supplier will be dealt with from a multitude of different roles.

The Supplier Contact directory allows the contact details of all these individuals to be associated with the Supplier Details, thus allowing the sharing of this information across the board from a single central location.

Click the Supplier Contact link will navigate the user to the contacts screen.
