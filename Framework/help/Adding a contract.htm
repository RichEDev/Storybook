Purpose; The key aim of this screen is to facilitate the pro-active management of contracts.

Structure; The contract is presented on a single screen with has six major segments:
This holds, for information purposes and automated email notifications of renewals, data on basic contract information e.g. Vendor Name, Contract Number, Cost, Type of Contract, Start, End & Review dates.
Details For Adding A Contract
  • SELECT Contracts from the Main Menu

  • SELECT Add Contract - Insert the Contractual Information.

  • SELECT Update when complete

    • Field Descriptions
    Description Field: A Description must be defined - Insert the description required.


    Vendor Field : A Vendor must be defined -From the drop down menu SELECT Vendor required.


    You only need to complete the remaining fields required specific to the contract type.


    If extra fields are required the undefined fields can be used.
    They can be set in either a character, date, numeric format or a pre-defined option list. These are defined at the logon section, by the Administrator. See Administrator Functions on page 17
