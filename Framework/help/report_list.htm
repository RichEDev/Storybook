Report List
Overview

The Report List screen shows a user all reports that are available to them.

Reports within Framework are split into two types, Personal and Global.

For added clarity, all reports have a Name (mandatory) and a Description (optional).

To execute a report: simply click the hyperlink (underlined) report name from the list.

There is no limit to the number of reports a user can create.

Note:Once a report has been run, this becomes the user 'active' report.

Once an active report is established, Select Columns, Criteria or View Current from the menu will apply to the active report.

This will remain the active report until either another report is selected from the report list or until the user logs out.

Personal Reports

Personal reports are those created by you for your sole use. These reports are not visible or accessible by any other user of Framework.

Each user can create any number of reports which can be recalled and reused at any time.

Global Reports

Any report created by a user can, if their permissions permit, publish a report for the use of all other Framework users.

Tip: To improve the flexibility of Global Reports, it is recommended that the report's criteria be configured for run-time user selection rather than fixed. Refer to the Criteria Help for more information about this
